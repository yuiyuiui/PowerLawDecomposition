# Part 8: Wynn Epsilon vs Matrix Pencil — 对比报告

## 1. 方法概述

### Wynn Epsilon 方法

Wynn epsilon 是一种序列加速算法，等价于 Padé 逼近。我们尝试用它提取指数和的衰减率：

**策略：比值序列 + 迭代剥离**

对于 f(x) = Σ cₙ exp(-aₙ x)，比值 rₖ = f(xₖ₊₁)/f(xₖ) 以几何速度收敛到 z₁ = exp(-a₁ h)（最慢衰减分量的特征值）。Wynn epsilon 加速此收敛：

1. 计算信号尾部的比值序列 rₖ
2. 对 [rₖ, rₖ₊₁, ...] 施加 Wynn ε → 得到 z₁ 的加速估计
3. 还原 a₁ = -log(z₁)/h
4. 估计 c₁，从信号中减去 c₁ exp(-a₁ x)
5. 对残差重复步骤 1-4，提取 a₂, a₃, ...

实现了三种变体：
- `wynn_peel`：从尾部（大 x）开始剥离慢分量
- `wynn_peel_both_ends`：尾部提取慢分量 + 头部提取快分量
- `wynn_ratio_grid`：在多个窗口位置运行 Wynn，聚类结果

### 2.2 逐分量分析（domain=[1,5], h=0.1, N=10）

```
wynn_peel:
  a[ 1] = 0.49903260  (true=0.5, err=9.67e-04)  ← 第一个分量提取准确
  a[ 2] = 0.49971225  (true=1.0, err=5.00e-01)  ← 又提取到 a≈0.5！
  a[ 3] = 0.49988895  (true=1.5, err=1.00e+00)  ← 依然是 a≈0.5
  ...
  a[10] = 0.50000169  (true=5.0, err=4.50e+00)  ← 全部坍缩到 a≁0.5
```

**致命缺陷：迭代剥离完全失败。** 第一个分量 a₁=0.5 提取精度不错（err=9.67e-04），但减去 c₁exp(-a₁x) 后的残差中，比值序列再次收敛到 a≈0.5 而非 a₂=1.0。

### 2.3 简单场景（domain=[0,20], h=0.1, N=5）

即使在信息充足的简单场景下：

```
wynn_peel:
  a[1] = 0.500 (err=8.4e-09)  ← 极好
  a[2] = 0.500 (err=5.0e-01)  ← 坍缩
  a[3] = 0.500 (err=1.0e+00)  ← 坍缩
  ...

matrix_pencil (delayed):
  a[1] = 0.500 (err=3.3e-16)
  a[2] = 1.000 (err=7.5e-13)
  a[3] = 1.500 (err=1.7e-11)
  a[4] = 2.000 (err=4.8e-11)
  a[5] = 2.500 (err=1.5e-11)
```

**Matrix Pencil 达到 10⁻¹¹ 精度，Wynn 在第二个分量就失败了。**

## 3. 失败原因分析

### 3.1 迭代剥离的灾难性误差放大

Wynn 方法的核心问题是**迭代剥离（iterative peeling）的数值不稳定性**：

1. **第一步**：从尾部提取 a₁ ≈ 0.5000（精度 ~10⁻⁴），估计 ĉ₁
2. **减法**：残差 g(x) = f(x) - ĉ₁ exp(-â₁ x)
3. **问题**：由于 â₁ 和 ĉ₁ 有 O(10⁻⁴) 的误差，残差 g(x) 在尾部（大 x）仍包含 O(10⁻⁴) 量级的 exp(-0.5x) 项
4. **后果**：残差的比值序列**再次被这个 O(10⁻⁴) 的残留 exp(-0.5x) 主导**，因为它衰减最慢
5. **坍缩**：第二轮 Wynn 又提取到 a ≈ 0.5

这是迭代剥离的本质缺陷——**每步的微小误差会在后续迭代中被放大**，因为我们总是在提取残差中衰减最慢的分量，而上一步的残留误差恰好衰减最慢。

### 3.2 与 Matrix Pencil 的根本差异

| 特性 | Wynn 迭代剥离 | Matrix Pencil |
|------|-------------|---------------|
| 提取方式 | 逐个串行，每步依赖前一步 | **一次性同时**提取所有 N 个分量 |
| 误差传播 | 指数级放大（残留误差主导后续提取） | 独立（SVD 分解各分量正交） |
| 信息利用 | 仅用比值序列（一维信息） | 利用完整 Hankel 矩阵（二维结构） |
| 适合场景 | 单分量或分量间衰减率差异极大 | **多分量、任意衰减率** |

### 3.3 Wynn 的理论局限

Wynn epsilon 本质上是一维序列加速器——它优化的是**一个极限值**的收敛速度。要从 N 个指数项的混合信号中同时提取 N 个参数，需要的是**矩阵分解**（如 SVD + 特征值），这是一维方法无法替代的。

## 4. 结论

| 指标 | Wynn Epsilon | Matrix Pencil |
|------|-------------|---------------|
| 单分量提取 | 优秀（~10⁻⁴ Float64） | 极优（~10⁻¹¹ Float64） |
| 多分量提取 | **完全失败**（迭代坍缩） | 优秀 |
| 计算复杂度 | O(N·M) | O(M·L + L³) |
| 实用性 | 仅适用于粗估单个衰减率 | **通用推荐** |

**Wynn epsilon 不适合用于多分量指数和分解。** 正如用户所预期的，Matrix Pencil 系列方法在这个问题上具有压倒性优势。Wynn 的价值在于序列加速和极限估计，而非结构化的参数提取。
